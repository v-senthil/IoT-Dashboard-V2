About Project

Robots are used for special applications like handling hazardous situations and tasks that need high accuracy and speed. To implement real time inspection and surveillance of the border and home security, intelligent remote monitoring system is developed. Autonomous Remote Monitoring System is designed using Raspberry Pi, Sensor Network, Database which gives the user an imminence experience in using our robot Car, which can be used in emergence situation.


Made using the best available opensource application and platform available. All cloud based which maked users easy to access data and the device from anywhere in the world in real time, fast, easy and quick. Taking advantage of GCP - Google's Firebase, Storage System and SQLite3 Database is used.

Dashboard

Responsive Dashboard which can not be obtained in any online IoT Framework. Simple, Elegent, and Precise.

Charts and Data

For easy understanding of data and the values Charts are used with respect to the sepecific time.

Tracking

GPS Tracking of the robot car is esstential and is also done line without any delay.

Cloud File Storage

Upload your files for future analysis and Visualisation securely in the cloud. Which can be accessed only by the admin

Cloud Data Storage

Google Sheet and Firebase Integraion gives limtless access to the data like Android Application, Cloud Services, etc.

Data Visualisation and Analysis

External Visualization tools like Tableau and Power Bi are used to visualize and analyse data.

Live-Stream

As Tracking is enabled, Live stream using Raspberry Pi Cam is useful which is embedded in the system it self.

Reports

All the data is stored in Database, which then can be downloaded as Excel and CSV format for further analysis.

Notification

If any sensor value exceeds the thershold value, the user will receive a message and an Email automatically (or) manually.

MIT App Inventor

To make it more user friendly and interactive, an Anddoid Application is made using MIT App Inventor where you can access data and location of the device.

Object Detection

Taking Advantage of the Raspberry Pi Camera, Object Detection is embedded to give the users more interaction and friedly usage of the device.

Data History

All the data collected in stored in the database, which is used to visualize sensor values seperatly.
